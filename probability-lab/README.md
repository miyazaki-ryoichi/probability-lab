# 確率・統計 可視化ラボ

授業で扱う確率・統計の主要トピックを、学生が操作しながら確認できる静的Web教材です。

## 起動方法

```bash
python3 -m http.server 8087
```

ブラウザで `http://127.0.0.1:8087/` を開きます。

## GitHub Pagesで公開する場合

この教材は静的ファイルだけで動くため、GitHub Pagesでそのまま公開できます。

1. このプロジェクトをGitHubリポジトリにpushする
2. GitHubのリポジトリ画面で `Settings` → `Pages` を開く
3. `Build and deployment` の `Source` を `Deploy from a branch` にする
4. `Branch` を `main`、フォルダを `/root` にして保存する
5. 数分後、次のURLで開く

```text
https://GitHubユーザー名.github.io/リポジトリ名/probability-lab/
```

授業で学生に配布する場合は、このURLをLMSや授業資料に貼れば使えます。

## 収録内容

- データと代表値: 度数分布表、ヒストグラム、平均、中央値、最頻値、分散、標準偏差
- 相関と回帰: 散布図、相関係数、回帰直線、外れ値の影響
- 条件付き確率: 2×2表、検査の例、ベイズの定理、人数モデル
- 二項分布とポアソン分布: 確率変数、二項分布、ポアソン分布、平均と分散、まれな事象の近似
- 二項分布と正規分布の関係: 正規近似、近似条件、連続性補正

数式表示にはMathJaxを使い、教材内の主要な式はLaTeX形式で記述しています。

## 授業での使い方

各タブの下部に「考察事項」を入れています。先に学生に予想を書かせてから、パラメータやデータを動かすと理解確認に使いやすくなります。
